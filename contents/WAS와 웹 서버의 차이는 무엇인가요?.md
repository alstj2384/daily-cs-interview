## WAS와 웹 서버의 차이는 무엇인가요?
### 정제한 답변
웹 서버는 html, js, css, 이미지 등의 정적 데이터를 내려줄 때 사용됩니다. 동적으로 처리가 필요한 요청은 WAS에 전달하여 요청과 응답을 중개하기도 합니다. 대표적인 웹 서버는 Apache와 Nginx가 있습니다. 
자바에서 WAS는 대표적으로 톰캣이 있는데요, 서블릿 컨테이너 기능을 제공하고, 동적 컨텐츠를 생성하거나 애플리케이션 로직을 처리하는데 특화되어 있습니다. 
정리하면 웹 서버는 정적 컨텐츠 처리에 특화되어있고, WAS는 동적 컨텐츠 및 데이터 처리에 특화되어 있습니다. 

정적 데이터를 WAS로 내려줄 수도 있지만, 웹 서버가 정적 데이터까지 제공하면 부담이 있을 수 있어, WAS는 중요한 애플리케이션 로직에 집중하고, 정적 데이터 제공은 웹 서버가 수행하여 분업하여 부담을 완화할 수 있습니다.
